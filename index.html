흰캡처

언제 어디서나 하나의 단축키로 모든 캡처 기능을

어디서나 무료이며, 프로그램 실행 중 광고가 없습니다.
복잡한 실행 화면이 없습니다.
무엇보다도, 빠르고 가볍습니다.


다운로드 문서 후원


- 흰캡처는 Windows 10에 최적화 되어있습니다.
- .NET Framework 4.7이 설치되어있어야 합니다.

 

 

 

 

특징

제한 없이

개인이든, 기업이든 무료로 제한없이 사용할 수 있습니다. 라이선스 걱정 없이 누구나 다운로드하실 수 있습니다. 프로그램을 잘 사용중일 경우 상단 '후원'메뉴를 통해 후원해주실 수 있습니다.

광고 없이

스폰서 프로그램, 광고 프로그램을 설치하지 않습니다. 또한, 프로그램 동작 중 팝업되는 광고도 없으며, 향후 업그레이드 또한 평생 무료입니다.

걱정 없이

프로그램 동작 중 사용자의 동의 없이 업데이트를 하지 않으며, 인터넷 연결을 하지 않습니다. 서버로 전송되는 데이터가 없으며, 오직 사용자의 컴퓨터에서만 동작합니다.

 

 

기능

단축키 방식

캡처 할 때마다 프로그램을 킬 필요가 없습니다. 흰캡처는 항시 실행중인 상태이기 때문입니다. 사용자 분들은 그저 캡처가 필요할 때, 캡처 단축키를 눌러주면 끝입니다.

자동 플로팅

캡처된 사진은 화면에 플로팅(오버레이)되어, 항상 최상단으로 고정됩니다. 플로팅 된 사진은 자유롭게 이동시킬 수 있으며, 사진을 삭제/저장 아이콘 위로 끌어서 삭제 혹은 저장할 수 있습니다.

높은 호환성

Windows 10에서 동작하는 경우, 사용자의 시스템 배율이 100%가 아니더라도 온전히 동작합니다. 4K 해상도에서 시스템 배율 200% 상태일 때 정상 동작됨을 확인하였습니다.

 

사진에 그리기

캡처한 사진을 우클릭하여 그리기 모드로 전환할 수 있습니다. 직선, 사각형, 펜 도구를 이용하여 간단한 표시를 사진에 할 수 있습니다. 강조할 부분은 강조하고, 가릴 부분은 가려서 공유해보세요.

간단한 동작

어떤 캡처는 이 버튼, 어떤 캡처는 저 버튼... 일일이 단축키를 외울 필요가 없습니다. "캡처하기" 단축키 하나로 캡처모드에 들어간 뒤, 상황에 맞는 캡처 방법을 골라 쓰세요.

꼼꼼한 피드백

흰캡처는 1인 개발 프로그램입니다. 그렇기에, 개발자가 직접 문의를 읽습니다.
자기 프로그램에 대해 가장 잘 아는 사람은 기획팀도, 운영팀도 아닌 개발자입니다. 제가 직접 문의 사항을 피드백해드립니다.
